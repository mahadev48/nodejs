3).What is NoSql Database?
Ans).NoSQL Database is used to refer a non-SQL or non relational database.
It provides a mechanism for storage and retrieval of data other than tabular
relations model used in relational databases.NoSQL database doesn't use tables 
for storing data.
It is generally used to store big data and real-time web applications.
